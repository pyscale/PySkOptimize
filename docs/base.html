class pyskoptimize.base.MLPipelineStateModel(*, model: pyskoptimize.base.SklearnTransformerModel, scoring: str, preprocess: List[pyskoptimize.base.FeaturePodModel] = None, postprocess: pyskoptimize.base.FeaturePodModel = None, targetTransformer: pyskoptimize.base.SklearnTransformerModel = None, cv: int = 5)

This represents the full pipeline state.

Here, we should have a scikit-learn model (i.e. Ridge, LogisticRegression) as the model parameter, the scoring metric that is supported in scikit-learn, the list of preprocessing steps across all of the features, the post process of the resulting features from the application of the preprocessing steps, and optionally a transformer model that will convert your target variable to the proper state of choice.

Variables

  • model – The sklearn transformer configurations for the model

  • scoring – The scoring metric

  • preprocess – The list of preprocessing applications on different features

  • postprocess – The list of postprocessing steps to apply onto the feature union

  • targetTransformer – The sklearn transformer to apply onto the target label before training the model

  • cv – The cross validation number
