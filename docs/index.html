Welcome to PySkOptimize’s documentation!

This software package focuses on enabling the generation of a Bayesian Optimized model from a configuration file.

This is the first attempt of developing a low-code methodology for Data Science & AI.

One notable issue is the inability to use integer uniform distributions.

Currently, the only solution is to use the categories structure, with the value of the categories being the list of all of the integers you want to consider.

More work will be done to resolve this.
